# PySimpleCalculator
A simple python calculator using PyQt6, nothing else to it!
# Usage
Go to releases and install the .exe file found there
# Prerequisites
You will need to use the following commands if not using the .exe file:
```
pip install simpleeval
pip install pyqt6
pip install sys
```
